# Sistema Web Centralizado de Cobro
**Proyecto de Tecnologías de Información II | Licenciatura en Informática | Universidad del Mar (UMAR)**

Este repositorio contiene el desarrollo inicial de un sistema web centralizado de cobro para un bar, propuesto como proyecto de la asignatura de Tecnologías de Información II.

## Descripción del Proyecto
El proyecto consiste en el diseño y desarrollo de una plataforma web orientada a mejorar la gestión de reservaciones y el proceso de cobro por consumo dentro de un bar. La propuesta busca centralizar estas operaciones para agilizar la atención, reducir tiempos de espera y mejorar la organización del servicio.

Actualmente, el proyecto se encuentra en una etapa inicial de desarrollo, por lo que se plantea como base una solución enfocada en:
- La gestión de reservaciones.
- El registro de consumos.
- La centralización del proceso de cobro.
- La mejora en la atención al cliente dentro del establecimiento.


## 🛠️ Tecnologías y Conceptos Relacionados
* **Desarrollo Web:** Plataforma web para la gestión operativa del bar.
* **Arquitectura:** Modelo cliente-servidor.
* **Base de Datos:** Almacenamiento de información de reservaciones, consumos y cobros.
* **Seguridad:** Control de acceso, autenticación de usuarios y protección de datos.
* **Comunicación:** Gestión centralizada entre el sistema y los usuarios del bar.

## 🎯 Objetivo General
Desarrollar un sistema web centralizado de cobro que permita gestionar reservaciones y consumos de manera eficiente dentro de un bar.
